# Voice Commands Reference

## English Commands

### File Operations
- "create file [name]" - Create a new file
- "open file [name]" - Open an existing file
- "save file" - Save current file
- "delete file [name]" - Delete a file
- "rename file to [name]" - Rename current file

### Code Generation
- "write a function for [description]" - Generate a function
- "create a class for [description]" - Generate a class
- "add a comment" - Add comment to selected code

### Code Analysis
- "explain this code" - Get code explanation
- "review this code" - Get code review
- "find bugs" / "fix bugs" - Debug assistance

### Navigation
- "go to line [number]" - Jump to line
- "search for [term]" - Search in code
- "next error" / "previous error" - Navigate errors

### Execution
- "run the code" - Execute current file
- "run tests" - Execute tests
- "build project" - Build the project

## Persian Commands (فارسی)

### عملیات فایل
- "فایل جدید بساز" - ایجاد فایل جدید
- "فایل رو باز کن" - باز کردن فایل
- "ذخیره کن" - ذخیره فایل فعلی
- "فایل رو پاک کن" - حذف فایل

### تولید کد
- "یک تابع بنویس برای..." - تولید تابع
- "یک کلاس بساز" - تولید کلاس
- "کامنت اضافه کن" - اضافه کردن کامنت

### تحلیل کد
- "این کد رو توضیح بده" - توضیح کد
- "کد رو بررسی کن" - بازبینی کد
- "باگ رو پیدا کن" / "باگ رو درست کن" - رفع اشکال

### اجرا
- "کد رو اجرا کن" - اجرای کد
- "تست رو اجرا کن" - اجرای تست‌ها

## Tips

1. Speak clearly and at a moderate pace
2. Wait for the recognition indicator before speaking
3. Use the confidence threshold in settings to adjust sensitivity
4. Vosk mode works offline, WebSpeech requires internet
